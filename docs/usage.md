# Использование API для предсказания риска сердечного приступа

Данный сервис позволяет загружать CSV-файл с медицинскими данными пациентов и получать предсказания о риске сердечного приступа (0 — низкий риск, 1 — высокий риск).

## Требования

- Python 3.8+
- Зависимости из `requirements.txt`

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <ваш-репозиторий>
   cd project-heart-risk-prediction
   python -m venv venv```
# Windows (PowerShell):
```.\venv\Scripts\Activate.ps1```
# Установите зависимости:
```pip install -r requirements.txt```
# Запустите сервер
```uvicorn src.app.main:app --host 127.0.0.1 --port 8001```
# Откройте интерфейс Swagger
http://127.0.0.1:8001/docs

Использование через API
Эндпоинты
GET /health — проверка работоспособности
Ответ: {"status": "ok"}
POST /predict — предсказание на основе CSV
Параметры:
Файл file (формат .csv)
Формат CSV:

Должен содержать колонку id
Все признаки из исходного датасета (27 колонок)
Пример структуры см. в data/heart_test.csv
